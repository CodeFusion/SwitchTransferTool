1.2.0

  • Improved Design - The official Switch app got a facelift, so STT did too! Header size has been significantly reduced and the transfer button has been moved to the bottom right.
  • Light Mode - Most developers may use dark mode, but I know there are some of you who prefer the light. Your device's theme will be used by default, but you can choose a theme in Settings.
  • Pinch to Zoom - You can now zoom in and out on the camera screen using a pinch gesture. Scan your TV from across the room without standing up.
  • Faster Connection - QR Scanning is now default. Press the WiFi icon on the scanning screen to switch to manual mode or change the default in settings.
  • Android 13 Support - The app now uses the new more specific file permissions and has a Material You compatible icon.
  • Source Code - Code is finally available! View, fork, or make a PR on Github.
  • Library updates - That should mean fewer crashes.
